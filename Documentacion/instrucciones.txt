[INSTRUCCIONES DE EJECUCION]

1) Importar los proyectos de la Api(EmaApi) y el Cliente(EmaClient). IMPORTANTE: Las APKs ya estan dentro de la API (EmaApi\files\apks).
2) Click derecho a cada proyecto para ejecutarlos (Opcion Run As) como Maven Verify. Despues, se pueden ejecutar desde main como Java Aplication. IMPORTANTE: Ejecutar primero la Api.
3) Una vez con el cliente en ejecucion, se pueden hacer las siguientes cciones:
	- [Usuarios]: Ventana para comprobar el registro y login.
	- [Buscar APKs]: Ventana para listar todas las APKs o para buscar por Titulo una en especifico.
	- [Crear APK]: Ventana para crear APKs con todos sus campos.
	- [Actualizar APK]: Ventana para modificar una APK, especificada por Titulo. Los valores disponibles a modificar son el titulo, autor, descripcion y la imagen.
	- [Eliminar APK]: Ventana para eliminar una APK concreta por Titulo.
	- [Descargar APK]: Ventana para descargar una APK especifica por Titulo.
	- [Calcular Hash]: Ventana para calcular el Hash tras poner el Titulo de la APK y otros datos como algoritmo y archivo local se podran calcular el Hash del Servidor, Hash del archivo local y despues se podran comparar ambos Hash.
